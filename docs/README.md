# docs

Documentation for this repo.

- [`spec/`](spec/) — dogfooded behavioral specs and conformance matrices for the five workflows and a subset of the skills (coverage is partial; see [`spec/README.md`](spec/README.md)). Each spec describes *what* and *why* (the contract); its sibling `<spec>.conformance.md` is a section-by-section audit (Conformed / Diverged / Not-built) of the spec against the actual implementation. All indexed specs are `implemented`.
